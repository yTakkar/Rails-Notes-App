# Usage
1. Open PHPMyAdmin, create a `DB` named `rails-notes-app` & import `db.sql` file.
2. Create the following environment variable:
```javascript
RAILS_DB_PASSWORD="DB PASSWORD"
RAILS_SOCKET="MySQL Socket"
```

5. Start the server
```javascript
rails s
```
6. Now, open it in browser:
```javascript
localhost:3000
```
